# LeetCode Practice Repository

This repository contains my solutions to LeetCode problems across multiple programming languages, focused on building strong problem-solving skills and deep understanding of algorithms and data structures.

---

## Structure

Problems are organized by:

* **Language** (Java, Kotlin, TypeScript)
* **Difficulty** (Easy, Medium, Hard)
* **Problem name**

Each problem has its own folder containing:

* One or more implementations (e.g., brute force and optimized)
* A `README.md` with explanation, approach, and complexity analysis

Example:

```
java/easy/two-sum/
├── SolutionHashMap.java
├── SolutionBruteForce.java
└── README.md
```

---

##  Goals

* Improve algorithmic thinking
* Understand trade-offs between different approaches
* Practice writing clean and readable code
* Build consistency through daily problem solving

---

##  Approach

For each problem:

* Start with a **brute force solution**
* Optimize step by step
* Analyze **time and space complexity**
* Document key insights and patterns
